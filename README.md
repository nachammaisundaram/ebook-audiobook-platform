E-Book & Audiobook Platform
A desktop application built with Python (Tkinter) that makes reading and 
listening more accessible — with PDF search, bookmarking, and a built-in 
text-to-speech engine.

Problem
Most e-book apps are either too basic or not accessible for users who prefer 
listening over reading. There was no simple offline tool combining both.

What I Built
A Python desktop app where users can read PDFs, search content, bookmark 
pages, and convert any e-book to audio using a custom TTS engine.

Tech Stack
- Language: Python
- UI: Tkinter
- Features: PDF viewer, TTS engine, bookmarking, search

Key Features
- PDF reading with search & highlighting
- Bookmark pages for quick access
- Text-to-speech with speed control & voice selection
- Audio playback with chapter navigation

Results
- 98% unit test coverage across all modules
- High accessibility for users with reading difficulties
